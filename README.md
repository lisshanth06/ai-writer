# FastHTML Hello World

A simple Hello World application using the FastHTML framework.

## Prerequisites
- Python 3.10+
- `uv` package manager

## Running the Application

1. Install the required dependencies using `uv`:
   ```
   uv sync
   ```

2. Run the application using `uv`:
   ```
   uv run python .\manage.py runserver
   ```

3. Open your browser and navigate to:
   ```
   http://localhost:8000
   ```

You should see a "Hello World" message displayed in blue.

## Note
This project uses `uv` for package management and running Python scripts instead of traditional `pip`. Make sure you have `uv` installed and configured correctly.
